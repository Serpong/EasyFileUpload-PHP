# File Upload Class 

usage
------- 

```php
<?php
	$fileUpload = new FileUpload();

	$fileUpload->setFile($_FILES['input_file']);
	if($fileUpload->checkFile())
		echo "this file is ok";
	else
		echo "this file is not ok";

	$fileUpload->upload();

	print_r($fileUpload->getInfo());
?>
```
